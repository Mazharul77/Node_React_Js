    • Promise then Fethch

    Open console in browser and understand the flow of operations.

    [Note: The fetch operation will only work if you have an active internet connection.]
    In this example, we create a new Promise that simulates an asynchronous operation using setTimeout. After 2 seconds, the promise is resolved with a message. We then use the .then() method to handle the resolved value and initiate a fetch operation to retrieve data from a placeholder API. If the fetch operation encounters an error (e.g., no internet connection), we catch it and log an error message. Finally, we handle the response from the fetch operation and log the fetched data to the console.
    Alternatively, we can use async/await for a more synchronous-looking approach. But in async await, we need to wrap our code inside an async function and use try/catch for error handling.
